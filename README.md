# GTMAgent

Autonomous AI agent system for go-to-market intelligence, competitive analysis, and marketplace operations.

## Stack

- **Backend**: Python (FastAPI)
- **AI**: OpenAI, Anthropic
- **Tools**: Apify, Exa, NVM/Morpheus marketplace
- **Deployment**: Render

## Setup

```sh
pip install -r requirements.txt
python src/main.py
```
